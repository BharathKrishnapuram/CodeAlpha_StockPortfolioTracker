# 📈 Stock Portfolio Tracker

A Python-based Stock Portfolio Tracker developed as part of the CodeAlpha Python Programming Internship.

## 📌 Description

This project helps users calculate their stock investments by selecting stocks and entering quantities. It provides a portfolio summary, identifies the highest investment stock, calculates portfolio distribution percentages, and saves the report to a text file.

## ✨ Features

- Multiple stock purchases
- Investment value calculation
- Portfolio summary
- Highest investment stock analysis
- Portfolio distribution percentage
- Error handling for invalid inputs
- Report generation using file handling

## 🛠️ Technologies Used

- Python
- Dictionaries
- Loops
- Conditional Statements
- File Handling
- Exception Handling

## 🚀 How to Run

1. Clone the repository

```bash
git clone https://github.com/your-username/CodeAlpha_StockPortfolioTracker.git
```

2. Open the project folder

```bash
cd CodeAlpha_StockPortfolioTracker
```

3. Run the program

```bash
python main.py
```
